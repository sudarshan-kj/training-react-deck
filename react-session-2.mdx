### Closures

Whenever you create a function within another function, you have created a closure.

---

<CodeSurfer>

```js
function outerFunction() {
  const outer = `I see the outer variable!`;

  function innerFunction() {
    console.log(outer);
  }

  return innerFunction;
}

outerFunction()();
```

</CodeSurfer>

# Shorter Version

---

<CodeSurfer>

```js
function outerFunction() {
  const privateVariable = `I see the outer private variable!`;

  return function innerFunction() {
    console.log(privateVariable);
  };
}

outerFunction()(); // I see the outer variable!
```

</CodeSurfer>

---

### Closures have access to the variables in the outer function

- Are used to :

  - To control side effects

  - To create private variables
